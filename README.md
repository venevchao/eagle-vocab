# 🦅 養鷹達人 — 家長指南

**App 網址：https://venevchao.github.io/eagle-vocab/**

養鷹達人是一個英文 sight words 每日練習 App：孩子每天練 10 分鐘，每個單字是一顆蛋，答對就逐步進化成雄鷹。全中文介面，設計給國小學童自行操作；免帳號、免安裝、免費，iPad／iPhone／電腦瀏覽器皆可使用。

---

## 第一次使用

1. 用孩子常用裝置的瀏覽器打開上方網址（建議 iPad Safari）。
2. 點選分享按鈕 → **加入主畫面**，之後就能像 App 一樣全螢幕開啟。
3. 進入練習後若沒有聲音，請先按實體音量鍵確認**媒體音量**沒有歸零，再檢查控制中心的靜音 🔕；發音使用系統內建語音，不需網路下載。
4. 第一次使用錄音功能時，系統會跳出英文的麥克風授權對話框，建議家長陪同按下「允許」。
5. 若每次進練習都重新詢問「是否允許麥克風」：點網址列**左側的圖示選單**（不同版本 Safari 圖示不同，找「網站設定」或「aA」都是它）→ 網站設定 → 麥克風 → 改成「允許」，之後就不會再詢問。
6. 家長想自己試玩：在網址後面加上 `?test=1` 會進入測試沙盒，資料完全獨立，怎麼玩都不影響孩子的進度。

## 孩子每天做什麼

打開 App 按下「開始今天的練習」，完成一場約 8–10 分鐘的練習即可，包含三種題型：

- **聽音拼字**：聽發音，用字母磚拼出單字。
- **看字唸出**：看單字大聲唸出來，再聽答案自我核對。
- **聽音選字**：聽發音，從四個選項中選出正確的字。

答錯的字會在同一場內再次出現，直到答對為止。每場結束有星星、進化動畫與徽章結算。當天達標後想再練是**自由練習**：鳥兒等級照算、星星不計，練習慾望不受壓抑，天梯速度也不會失控。

## 錄音與口說（全部自由參加，絕不評分）

- **🎙️ 我要錄音**：「看字唸出」時可以錄下自己的發音，聽答案時和標準音來回比對（Echo 跟讀法）。
- **🎙️ 挑戰造句 +5⭐**：用這個字自由說一句話，錄下來聽——每個字每場一次。
- **🦅 雄鷹句子任務 +5⭐**：畢業的字偶爾回鍋時，會變成整句跟讀卡。
- **🕊️ 鳥園自由飛行區**：點鳥園裡任何一隻有例句的鳥，隨時自由跟讀例句，不計分、不影響進度，想練幾次都可以。

三個設計原則，家長可以放心：

1. **絕不評分發音**——只提供「聽自己 ↔ 聽標準音」的回放比對，孩子自己聽出差異。
2. **錄音絕不上傳、絕不保存**——音檔只暫存在記憶體，換卡即丟，App 沒有任何後台。
3. **錄音永遠是加分項**——不錄音也能完整走完所有練習，任何錄音步驟都不會擋路。

小提醒：按下錄音鈕會先顯示「🎙️ 準備中…」，約 1–3 秒後自動轉為「⏺️ 錄音中」才開始收音——這是 iPad 啟動麥克風的系統延遲，不是故障，等它變紅再開口即可。

## 養鷹規則書（蛋 → 雄鷹）

每隻鳥代表一個單字，階段就是記憶熟練度的視覺化：

| 階段 | 進入條件 | 下次出題間隔 |
| --- | --- | --- |
| 🥚 蛋 | 剛匯入、還沒學過的字（家長一匯入即出現在蛋區） | 當天 |
| 🐣 孵化 | 經過「新朋友」介紹卡＋第一次答對 | 1 天後 |
| 🐤 雛鳥 | 到期複習再答對 1 次 | 2 天後 |
| 🐦 小鳥 | 再答對 1 次 | 4 天後 |
| 🦜 成鳥 | 再答對 1 次 | 8 天後 |
| 🦅 雄鷹 | 再答對 1 次（＝跨 5 個不同日子、共 5 次成功回想） | 16 天後仍會偶爾回鍋維護 |

三條關鍵規則：

1. **只有「到期複習的第一次作答」算數。** 同場答錯後的重試答對不升級——升級必須跨日，因為記憶要隔夜才算數。
2. **答錯降一級**（最低降到孵化，不會退回蛋），且當天內會再出現重練。
3. **雄鷹也會退位。** 畢業的字偶爾會被抽回快問快答，答錯會降回成鳥——熟練也會生鏽，維護是設計的一部分。

一個新字最快 15 天畢業（第 0、1、3、7、15 天各答對一次）；「認識」目標的字到成鳥即畢業，最快 7 天。

## 連續天數與鷹芒勳章 🎖️

- 每天完成一場練習，連續天數 🔥 加一。
- **每連續 5 天自動獲得一枚鷹芒勳章**（最多存 5 枚）。
- **落練一天會自動消耗一枚勳章，連續紀錄不中斷**；勳章不夠抵時紀錄歸零，但最高紀錄會保留顯示。
- 缺席回來沒有任何懲罰：待複習的字會分散到之後幾天消化（每場複習上限 30 題），複習債偏高時系統自動暫停發新字。

## 星星與馴鷹師天梯

星星 ⭐ 是孩子的努力帳本：第一次答對 +10、重試答對 +5、⚡ 拼字秒殺（又快又準）+2、造句挑戰與雄鷹句子任務各 +5。累積星星沿著十階「馴鷹師天梯」升級（鷹兒保育員 → 蒼穹鷹王），完整規則孩子可以在「祕笈」分頁自己查——那是寫給他看的規則書，建議讓他自己探索。

## 家長頁操作

點下方「家長」分頁：

- **學習狀況**：連續天數、勳章數、畢業字數、今日實際待複習量（孩子看到的永遠是修剪過的場次，這裡才是真實總量）、拼字順暢度、近 7 天練習量圖表。
- **頑固字排行**：由答錯、拼字失手、退格與速度合成的排行——最適合當親子閒聊、造句遊戲的素材，點字可聽發音。
- **媽媽的悄悄話**：寫幾句自訂的鼓勵話，會和內建加油語一起在首頁隨機出現；內容會跟著進度備份一起帶走。
- **新增單字**：把單字貼進輸入框（逗號、空格、換行分隔皆可），選擇目標層級後匯入：
  - **熟用**——要求完整產出（拼字＋口說），適合課內要求精熟的字；
  - **熟悉**——溫和起步、逐步過渡到完整產出，適合本學期新字；
  - **認識**——只練聽音辨識，門檻較低，適合先混個臉熟的擴充字。
- **進度備份**：練習進度儲存在**該裝置的瀏覽器內**。換裝置、清除瀏覽器資料前，請先備份：
  - **📤 匯出成檔案**（建議）——按下後 iPad／iPhone 會跳出系統分享面板，可選「儲存到檔案」（存進與家人共享的 iCloud 資料夾最方便，會自動同步到其他裝置）或直接用郵件寄出；Android 可選擇儲存或用 Gmail 寄送；電腦瀏覽器按了會直接下載存檔。
  - **匯出進度**（剪貼簿制，備用）——文字貼到訊息、備忘錄等處保存，換裝置時到「匯入進度」貼回即可。
  - 建議養成每日或每週備份一次的習慣，別讓資料只留在孩子的裝置上。
- **語音診斷**：遇到無聲問題時的排查工具——分開測試發音與音效、顯示語音紀錄，把結果截圖給 Claude 即可遠端判讀。
- 頁面最底部有**版本標記**：回報問題時附上這個日期，可以快速確認孩子裝置上跑的是哪一版。

## 設計依據（給想深究的家長）

出題與升級規則基於記憶科學的三個實證原理：提取練習效應（主動回想勝過重複閱讀）、間隔效應（在快忘記時複習效果最好，對應蛋→雄鷹的 1／2／4／8／16 天間隔）、以及二語詞彙的接受性／產出性不對稱（「看得懂」不會自動變成「用得出來」，所以題型以拼字與口說等產出練習為主）。同時為兒童動機做了調整：成功率維持約八成、答錯不重罰、每場以簡單題收尾。
